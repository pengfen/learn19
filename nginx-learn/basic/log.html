http log 模块 (http_log_module) http_log_module配置 syntax:; access_log path [format [buffer=size | off ]] #语法 default: log_format combined "..." #默认值 context: http, server #作用域 log_format gzip '$remote_addr-$remote_user[$time_local]' :'$request$status $bytes_sent' :'" $ http _ referer" "$http_user_agent" "$gzip_ratio"'; access_log /var/logs/nginx/access.log gzip buffer=32k; access_log 确定了 nginx 是否保存访问日志。将这个设置为关闭可以降低磁盘 IO 而提升速度。 error_log 设置 nginx 应当记录临界错误。
