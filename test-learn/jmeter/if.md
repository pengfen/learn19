# 测试如果(if)控制器

* 添加线程组 Test Plan(测试计划) ---> Add ---> Threads(Users) ---> Thread Group
* 添加自定义变量 Add ---> Config Element ---> User Defined Variables
* 设置变量 IfRun 1
* 添加If Controller Add ---> Logic Controller ---> If Controller
* If Controller上添加HTTP Request  Add ---> Sampler ---> HTTP Request
* 添加结果树  Add ---> Listener ---> View Results Tree
